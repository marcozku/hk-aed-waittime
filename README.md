# 香港急症室等候時間顯示系統

這是一個實時顯示香港各醫院急症室等候時間的網頁應用程式。

## 功能特色

✅ **實時數據** - 直接從醫院管理局API獲取真實數據，無模擬數據
✅ **自動刷新** - 成功連接後每15秒自動更新數據
✅ **智能重試** - 連接失敗時自動重試直到成功
✅ **地理定位** - 根據用戶位置顯示最近的醫院
✅ **多重排序** - 支援按距離、等候時間或醫院名稱排序
✅ **分類篩選** - 可按醫院聯網或地區篩選
✅ **天氣資訊** - 顯示當前天氣及特殊天氣警告
✅ **色彩編碼** - 根據國際標準用顏色顯示等候時間嚴重程度
  - 🟢 綠色: < 1小時
  - 🟡 黃色: 1-2小時  
  - 🟠 橙色: 2-4小時
  - 🔴 紅色: > 4小時
✅ **醫院詳情** - 顯示電話、地址、專科警告
✅ **快速操作** - 一鍵撥打電話或查看地圖
✅ **連接狀態** - 底部顯示實時連接狀態

## 使用方法

### 本地運行

1. 將所有文件放在同一目錄下
2. 使用瀏覽器打開 `index.html` 文件
3. 允許瀏覽器獲取您的位置（可選）
4. 等待數據加載完成

### 使用本地伺服器（推薦）

```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000
```

然後在瀏覽器中訪問 `http://localhost:8000`

## 數據來源

- **急症室數據**: 香港醫院管理局開放數據
- **天氣數據**: 香港天文台開放數據API

## 醫院資料核查

本系統已核查以下醫院專科資料：

- **北區醫院 (NDH)**: ⚠️ 沒有兒科、婦產科、神經外科住院服務
- 其他醫院提供完整急症服務

## 瀏覽器支援

- Chrome/Edge (推薦)
- Firefox
- Safari
- 需要支援 ES6+ JavaScript

## 注意事項

- 首次加載時會請求地理位置權限（用於計算距離）
- 如拒絕位置權限，將使用香港天文台位置作為默認位置
- 需要網絡連接才能獲取實時數據
- 數據更新頻率取決於醫管局API更新頻率

## 技術細節

- 純前端實現，無需後端服務器
- 使用 Fetch API 進行數據獲取
- 響應式設計，支援手機和桌面設備
- 使用 Haversine 公式計算地理距離
